Step 1: Cek Apps Script URL

Pastikan URL Apps Script sudah benar dan Web App sudah deployed:

https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec

Step 2: Cek Deployment Settings

Di Apps Script, pastikan deployment settings:

  • Execute as: Me
  • Who has access: Anyone
  • Status: Active

Step 3: Cek Google Sheet

Pastikan Google Sheet:

  • Sheet ID sudah benar di Apps Script CONFIG
  • Sheet accessible (not private)
  • Apps Script punya permission ke sheet

Step 4: Test Manual

Test manual di browser:

YOUR_APPS_SCRIPT_URL?action=getCOA

Harus return JSON response dengan success: true
